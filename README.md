
# YouTube Data Analysis 

## Overview
This project performs Exploratory Data Analysis (EDA) on YouTube video data
using Python and Jupyter Notebook to understand user engagement metrics.

## Key Analysis
- Correlation between views, likes, and dislikes
- Heatmap visualization
- Engagement interpretation

## Tools & Libraries
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- SQLAlchemy
